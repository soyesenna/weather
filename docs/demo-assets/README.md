# Demo assets

- `seoul-flood-demo.gif`: 30초 시연 흐름 GIF. 시민 요약 → 지도 레이어 → 안전경로 → 익명 제보 → 운영자 콘솔 순서로 구성.
- 생성 명령: `python3 -m venv /tmp/weather-gif-venv && /tmp/weather-gif-venv/bin/pip install pillow && /tmp/weather-gif-venv/bin/python scripts/generate-demo-gif.py`

실제 Vercel 배포 URL이 확정되면 브라우저 녹화본으로 교체 가능하다.
